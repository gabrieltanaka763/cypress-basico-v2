# Testes automatizados com Cypress - Básico

👋 Seja bem-vindo(a)!

É muito bom tê-lo(a) aqui. Tenho certeza que você vai adorar este curso. ❤️

## Pré-requisitos

- Google Chrome
- Git
- Visual Studio Code
- Npm (Node.Js) 

## Como rodar ?

No terminal:
Rodar simulando desktop:
 - npm run cy:open
 - npm run cy:test

Rodar simulando mobile:
- npm run open:mobile
- npm run test:mobile
